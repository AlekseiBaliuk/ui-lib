# UI Library

A simple UI component library built with **Next.js** and **Tailwind CSS**.
This project demonstrates different UI components and their variants in a clean and organized interface.

## ✨ Features

* Modern UI built with **Next.js**
* Styled using **Tailwind CSS**
* Component examples and variants
* Responsive layout
* Sidebar navigation for browsing components

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/AlekseiBaliuk/ui-lib.git
cd ui-lib
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the app in your browser:

```
http://localhost:3000
```

## 📁 Project Structure

```
app/
  layout.tsx      # Root layout
  page.tsx        # Main page
components/       # UI components
public/           # Static assets
styles/           # Global styles
```

## 🧩 Components

This project will include examples of reusable UI components such as:

* Buttons
* Inputs
* Cards
* Modals
* Navigation elements

Each component may include multiple **variants and states**.
